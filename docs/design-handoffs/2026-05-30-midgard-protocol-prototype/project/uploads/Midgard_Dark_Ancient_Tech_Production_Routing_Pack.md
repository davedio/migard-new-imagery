# Midgard Dark Ancient Tech Production Routing Pack

Prepared: 2026-05-28

Status: Tool-by-tool production routing for the Dark Ancient Tech / Mythic Tree World system.

## Non-Negotiable Direction

The active production model is Mythic Tree World with Dark Ancient Tech styling and trust architecture as the mechanism layer. It is not a crude stick-frame tree, literal forest, colorful ecosystem mural, or older Tree World handoff.

The ancient-tech world can still be tree-themed. The key is execution: tree DNA should read as architecture, topology, circuitry, and settlement structure rather than a flat tree drawing.

The active model is:

- Stone-and-gold proof gate.
- Ancient trust engine.
- Architectural world-tree topology.
- Organic Rootwork as technical topology.
- Settlement bedrock rail.
- Dark technical architecture.
- Gold proof circuitry.
- Controlled Midgard green verification signal.
- Deep ink, obsidian, basalt, charcoal, muted gold, and precise labels.

Only inherited visual asset by default:

- Base Midgard logo.

Everything else should be newly produced for this direction.

## Recommended Production Order

1. Claude Design creates 3 ancient-tech tree/proof-gate directions and 3 living-system directions.
2. Human designer selects one visual route and rejects anything that looks like a stick-frame tree, literal forest, or generic crypto.
3. Figma turns the winner into a design system: colors, type, components, safe zones, deck frames.
4. Image model or Claude Design produces cleaned still candidates for proof gate, settlement rail, Secure Migration, and proof cards.
5. Human designer cleans the winning stills and removes artifacts.
6. Kling tests subtle motion only from cleaned stills.
7. Rive builds the Trust Layers explainer from the Figma state map.
8. Canva receives locked templates only after Figma and art direction are approved.
9. Codex implements the web prototype and regenerates PPTX/code assets from approved frames.
10. Human designer and Codex run final QA.

## Acceptance Standard

A candidate passes only if:

- Any generated copy follows Category Rebel / Confident Challenger voice: sharp thesis, plain-spoken mechanism, 70 direct / 30 defiant, no cheap shots.
- It reads as dark ancient technology at first glance.
- Gold is visibly part of the trust/proof system, not a tiny accent.
- If it uses tree-world DNA, it reads as architectural topology, not a literal tree illustration.
- It has no stick-frame tree, literal forest, wooden trunk, leafy canopy, or fantasy nature scene.
- It has no generic crypto signals: coins, chains, cubes, token icons, blue-purple gradients.
- It has no fake text or pseudo-UI from an image model.
- It can support real Midgard logo, proof labels, source/status chips, and responsive crops.
- It still works as a static image before motion.
- It has a clear reduced-motion or fallback path.
