# Security

Midgard Gateway is the public website and status surface for Midgard. It should never ask for private wallet secrets, seed phrases, recovery phrases, private keys, or unnecessary wallet approvals.

## Reporting

Report suspected vulnerabilities to `info@anastasialabs.com`. The current reporting policy is
available at `/security-policy`; preserve evidence and include the details below.

Useful evidence includes:

- URL or account name,
- screenshot,
- timestamp,
- browser or wallet context,
- steps to reproduce,
- whether any wallet interaction occurred.

Do not include private keys, seed phrases, recovery phrases, or wallet passwords in a report.
